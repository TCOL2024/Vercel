# Linda – Lernassistentin mit FSRS4JS

## Features
- KI-gestützte Lernassistentin
- Lernkarten mit wissenschaftlicher Spaced Repetition (FSRS4JS)
- Mehrere Fachmodi: AEVO, VWL, Personalwesen
- Mobile-optimierte Oberfläche
- Offline-Fähigkeit
