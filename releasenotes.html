Latest Releases (Jan 2026)

Linda 2026 · Release 1 14. Jan 2026 Mobile Beta, Performance-Verbesserungen, Modell-Upgrade (teilweise GPT-5.1)
New Features Performance Beta

New Features

  • Linda Mobile 2026 (Beta): Smartphone-optimiert, weniger UI-Ballast, kürzere Ladezeiten.
  • Separate mobile URL/Ansicht mit identischer Antwortlogik, aber reduziertem Look.

Resolved Issues / Improvements

  • Performance-Tuning: schnellere Antworten durch effizientere Requests/geringere Payload.
  • Teilweise Upgrade auf GPT-5.1 (je nach Setup), um Qualität/Tempo zu verbessern.

Security Notes

  • Empfehlung/Architektur: Client ruft Vercel-Proxy statt Make-Webhook direkt auf (Endpoint-Schutz).

Notes

  • Kommunikationshinweis: KI kann Fehler machen; gutes Prompting bleibt wichtig.
  • „Lernen“/„Fachmodus“: noch nicht final enthalten (verschoben).
Linda Secure · Hardening Sprint 07.–13. Jan 2026 Security-Analyse, Prompt-Injection-Risiko, Proxy-Stabilisierung
Security Governance Proxy

Security Notes

  • Prompt-Injection als inhaltliches Risiko: nicht rein durch Frontend (HTML/JS) lösbar.
  • „System-Prefix + User-Text zusammenkleben“ ist angreifbar → Guardrails/Policy-Design erforderlich.
  • „Red-Team“-Testdenken (gezielte bösartige Eingaben) als Qualitätsstandard.

Resolved Issues / Improvements

  • Arbeit an Proxy-Funktionen (z. B. PROXY_LINDA2026) und Weiterleitung zu Make.
  • Fokus auf saubere Request-Formate (Method/Headers/Body) + Logging als Debug-Basis.
Linda Mobile UI · Iteration 04.–15. Jan 2026 Mobile Lesbarkeit, iMessage-Feeling, Share/Print-Wünsche
New Features UX

New Features

  • Share-Idee: „Kopieren oder via WhatsApp/Mail teilen“ als Quick-Action.
  • Print-Idee: Schriftart Aptos, Frage im Ausdruck leicht blau hinterlegt.

Resolved Issues / UX Improvements

  • Schrift & Chat-Lesbarkeit: Feedback „zu klein“ → Typo/Spacing hochsetzen.
  • Composer/Input: Feedback „zu klein“ → deutlich größere Eingabe-Zone.
  • Neuer Chat leichter erreichbar; Avatar mobil optional/entfernt.
  • Quellen-Darstellung „schöner“ (statt unschöner Bullet-Optik).

Known Issues & Hinweise

  • Prompt-Injection: Nicht durch „Frontend-Tricks“ lösbar. Du brauchst serverseitige Policy/Guardrails (und am besten ein sauberes Rollen-/Kontextmodell).
  • Proxy → Make: Wenn „nichts ankommt“, sind die üblichen Ursachen Method/Headers/Body-Encoding, CORS/Preflight, Timeouts und fehlendes Logging.
  • Roadmap-Lücke: „Lernen“/„Fachmodus“ sind angekündigt, aber noch nicht final (Stand Jan 2026).
